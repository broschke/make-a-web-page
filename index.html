Lesson One

Major pieces of the web
Browsers - a program that runs on your computer to display files found on the web
The internet - the world's largest computer network
HTTP - protocol to connect browser to server
Servers - computers the host the files that make up the internet

HTML Markup
Consists of Tags and Content.
Tags tell the computer to how display the contents, together this is called an Element.
Format is <name> contents </name> or <opening tag> contents <closing tags>

Examples of tags
The <b> tag makes content bold
The <em> tag makes content italicised
The anchor tag, <a>, make links in a webpage. Ex: <a href="www.udacity.com">Udacity</a>
where href is an attribute and www.udacity.com is the value. Udacity,
The image tag format is as follows:
<img src="url" alt="text">
where url is where the image is downloaded from and the alt, or alternate, is the text that is displayed when the image fails to load.
Image tags have no closing tag because it has no content. These are also called void tag.

Whitespace tags
The <br> tag creates a line break, which moves text after the tag to a new line. The <p> tag renders content in one paragraph. Since it has content, it requires a closing tag, while the <br> tag is a void tag.
HTML code does not need to follow the action of the whitespace tags
Inline vs Block - the br tag is an inline element, simply text.
The <p> tag is a block element, an invisible block is created around the content of this tag
Span and Div elements
Simply contain their content but have the ability to edit how the content is laid out with CSS
Span is inline while div is block. Both require closing tags.

Lesson Two

A webpage is like a house - rooms are partitioned in a given manner, they each serve a purpose and can have its own unique style.
HTML - defines the parameters of the page
CSS - provides styling options
Javascript - creates interactivity

Document Object Model, or DOM, is the tree like structure that HTML block elements form.
HTML tags create elements that the DOM uses to display the page.
Webpages are constructed with boxes to simplify structure and organization.

Lesson Three

Cascading Style Sheet (CSS)
- Code that controls the style or appearance
- Can be applied to all element types in an html document or by selecting specific classes
- This avoids repetition of coding

There are three ways to include CSS in a webpage:
1. Write CSS in the head of html
2. Link html to a separate CSS files
3. Write style inline with html
